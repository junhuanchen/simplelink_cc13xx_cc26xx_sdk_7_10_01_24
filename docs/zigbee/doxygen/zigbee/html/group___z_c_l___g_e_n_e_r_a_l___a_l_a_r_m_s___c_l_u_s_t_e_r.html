Detailed Description

Macro Definition Documentation

§ ATTRID_ALARMS_ALARM_COUNT

#define ATTRID_ALARMS_ALARM_COUNT   0x0000

The AlarmCount attribute is 16 bits in length and specifies the number of entries currently in the alarm table.

§ COMMAND_ALARMS_GET_ALARM

#define COMMAND_ALARMS_GET_ALARM   0x02

This command causes the alarm with the earliest generated alarm entry in the alarm table to be reported in a get alarm response command

§ COMMAND_ALARMS_GET_ALARM_RESPONSE

#define COMMAND_ALARMS_GET_ALARM_RESPONSE   0x01

The get alarm response command returns the results of a request to retrieve information from the alarm log, along with a time stamp indicating when the alarm situation was detected.

§ COMMAND_ALARMS_RESET_ALARM_LOG

#define COMMAND_ALARMS_RESET_ALARM_LOG   0x03

This command causes the alarm table to be cleared, and does not have a payload.
