Detailed Description

Macro Definition Documentation

§ ATTRID_TIME_DST_END

#define ATTRID_TIME_DST_END   0x0004

The DstEnd attribute indicates the DST end time in seconds. The value 0xffffffff indicates an invalid DST end time.

§ ATTRID_TIME_DST_SHIFT

#define ATTRID_TIME_DST_SHIFT   0x0005

The DstShift attribute represents a signed offset in seconds from the standard time, to be applied between the times DstStart and DstEnd to calculate the Local Time.

§ ATTRID_TIME_DST_START

#define ATTRID_TIME_DST_START   0x0003

The DstStart attribute indicates the DST start time in seconds. The value 0xffffffff indicates an invalid DST start time.

§ ATTRID_TIME_LAST_SET_TIME

#define ATTRID_TIME_LAST_SET_TIME   0x0008

The LastSetTime attribute indicates the most recent time that the Time attribute was set, either internally or over the network (thus it holds a copy of the last value that Time was set to).

§ ATTRID_TIME_LOCAL_TIME

#define ATTRID_TIME_LOCAL_TIME   0x0007

The Local Time is given by the equation: Local Time = Standard Time + DstShift (if DstStart <= Time <= DstEnd) Local Time = Standard Time (if Time < DstStart or Time > DstEnd)

§ ATTRID_TIME_STANDARD_TIME

#define ATTRID_TIME_STANDARD_TIME   0x0006

The local Standard Time is given by the equation: Standard Time = Time + TimeZone

§ ATTRID_TIME_TIME

#define ATTRID_TIME_TIME   0x0000

The Time attribute is 32 bits in length and holds the time value of a real time clock.

§ ATTRID_TIME_TIME_ZONE

#define ATTRID_TIME_TIME_ZONE   0x0002

The TimeZone attribute indicates the local time zone, as a signed offset in seconds from the Time attribute value.

§ ATTRID_TIME_VALID_UNTIL_TIME

#define ATTRID_TIME_VALID_UNTIL_TIME   0x0009

The ValidUntilTime attribute indicates a time, later than LastSetTime, up to which the Time attribute MAY be trusted.
