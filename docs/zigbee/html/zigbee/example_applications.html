Running the Example Applications¶

This section provides an overview of the Z-Stack out-of-box example applications and instructions on how to run them. The examples are out-of-box compliant with the ZCL v7 Specification but will further Product Certification will be required.

Note

Most examples have a coordinator, end device and router application. Green Power Device examples complement the Sink coordinator/router examples. The Network Processor is standalone. Generic App provides the template code for application development with minimal overhead (i.e. no ZCL support included). Sample App incorporates the Zigbee Application Builder in SysConfig to help configure the required Zigbee application device type, clusters/attributes, commands, and command callback APIs.

Note

All sample applications are supported for IEEE 802.15.4 2.4 GHz capable devices. The READMEs listed below point to the CC26X2R1 example but if a different platform is intended to be used, replace the board name in the path to find the corresponding document (e.g. “examples/rtos/CC1352R1_LAUNCHXL/zstack/zc_doorlock/README.html” if CC1352R1 platform is intended). Switch on-chip OTA solutions are also available for CC13x2x7 or CC26x2x7 devices.
