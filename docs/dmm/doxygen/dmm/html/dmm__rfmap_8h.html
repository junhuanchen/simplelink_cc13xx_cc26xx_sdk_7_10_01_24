dmm_rfmap.h File Reference

Detailed Description

Dual Mode Manager RF API remapping.

The dmm_rfmap header file remaps select functions in the RF driver API the DMM equivalents that are defined in the dmm_schedular module so that priorities and timing can be set according to the dual mode manager policy

#include <ti/drivers/rf/RF.h>
#include <dmm/dmm_scheduler.h>
