97 /*
98  * ========= SHA2 Secure Message Structs =========
99  * These secure message structs correspond to the secure message types defined
100  * above. Together, they are used by non-secure client to make PSA calls to the
101  * SHA2 secure service. There is a single input vector for the PSA call
102  * which is a pointer to secure message struct. If the underlying function
103  * has a return value, there is a single output vector which is a pointer to
104  * storage for the return value.
105  */
