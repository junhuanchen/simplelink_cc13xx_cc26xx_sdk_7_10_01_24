UART2CC26X2.h File Reference

Detailed Description

UART driver implementation for a CC26X2 UART controller.

============================================================================

The UART header file should be included in an application as follows:

#include <ti/drivers/UART2.h>
#include <ti/drivers/uart2/UART2CC26X2.h>

Refer to UART2.h for a complete description of APIs and examples of use.

Unsupported Functionality

The UART2 driver is unable to access flash memory in the address range 0x0000 - 0x2000 on devices based on the Cortex M33+ core (CC26X3/CC26X4) due to security constraints.

enum UART2CC26X2_FifoThreshold

UART TX/RX interrupt FIFO threshold select.

Defined FIFO thresholds for generation of both TX interrupt and RX interrupt.

Note
UART2CC26X2_FIFO_THRESHOLD_1_8 should be avoided as RX FIFO threshold as it can lead to unreliable read operations when using UART2_ReadReturnMode_PARTIAL
