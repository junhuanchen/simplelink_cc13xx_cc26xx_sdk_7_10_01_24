101 /*
102  * ========= AES CTR Secure Message Structs =========

104  * above. Together, they are used by non-secure client to make PSA calls to the
105  * AES CTR secure service. There is a single input vector for the PSA call

107  * has a return value, there is a single output vector which is a pointer to
108  * storage for the return value.
