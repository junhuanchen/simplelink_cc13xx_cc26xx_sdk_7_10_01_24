§ ToString() [2/3]

void ToString ( char *  aBuffer,
uint16_t  aSize 
) const

This method converts a given IPv6 socket address to a human-readable string.

The IPv6 socket address string is formatted as "[<ipv6 address>]:<port>".

If the resulting string does not fit in aBuffer (within its aSize characters), the string will be truncated but the outputted string is always null-terminated.

Parameters
[out]aBufferA pointer to a char array to output the string (MUST NOT be NULL).
[in]aSizeThe size of aBuffer (in bytes).
