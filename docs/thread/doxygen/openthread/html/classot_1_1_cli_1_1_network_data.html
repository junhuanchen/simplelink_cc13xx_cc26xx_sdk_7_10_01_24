Field Documentation

§ kFlagsStringSize

constexpr uint16_t kFlagsStringSize = 17
static

This constant specifies the string size for representing Network Data prefix/route entry flags.

BorderRouter (OnMeshPrefix) TLV uses uint16_t for its flags and ExternalRoute uses uint8_t, though some of the bits are not currently used and reserved for future, so 17 chars string (16 flags plus null char at end of string) covers current and future flags.
