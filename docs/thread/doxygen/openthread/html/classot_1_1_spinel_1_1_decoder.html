§ OpenStruct()

otError OpenStruct ( void  )

This method opens a struct in the frame.

After a successful call to this method, all the subsequent Read{SomeType}() methods decode and read the field/value from the current open struct until the struct is closed using CloseStruct() method. Structures can be nested. Up to kMaxNestedStructs nested structs can be opened at the same time.

OT_ERROR_NONESuccessfully opened a struct.
OT_ERROR_PARSEFailed to parse/open a struct.
OT_ERROR_INVALID_STATEAlready at the maximum number of nested open structures.

§ SavePosition()

void SavePosition ( void  )

This method saves the current read position in the frame.

A subsequent call to SavePosition() will overwrite the previously saved position. The saved position can be used to move the read position back (using ResetToSaved()) and re-read the same content.

Saved position can be within an open struct, and it remembers its enclosing struct. When the enclosing struct is closed, the saved position will be voided and can no longer be used. This ensures that we cannot jump back to middle an already fully decoded/read and closed struct.

References Decoder::mEnd, Decoder::mIndex, Decoder::mNumOpenStructs, Decoder::mSavedEnd, Decoder::mSavedIndex, and Decoder::mSavedNumOpenStructs.
