§ OPENTHREAD_CONFIG_DATASET_UPDATER_DEFAULT_DELAY

#define OPENTHREAD_CONFIG_DATASET_UPDATER_DEFAULT_DELAY   30000

Specifies the default delay (in ms) used by Dataset Updater when not included in Dataset already.

§ OPENTHREAD_CONFIG_DATASET_UPDATER_DEFAULT_RETRY_WAIT_INTERVAL

#define OPENTHREAD_CONFIG_DATASET_UPDATER_DEFAULT_RETRY_WAIT_INTERVAL   1500

Specifies the default retry wait interval (in ms) for Dataset Updater to wait (in addition to Dataset delay) after sending MGMT Set Pending Dataset command waiting for Active Dataset to be updated before retrying again before retrying the MGMT Set Pending Dataset command.
