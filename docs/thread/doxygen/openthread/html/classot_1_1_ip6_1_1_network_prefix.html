Detailed Description

This class represents the Network Prefix of an IPv6 address (most significant 64 bits of the address).

Member Function Documentation

§ GenerateRandomUla()

Error GenerateRandomUla ( void  )

This method generates and sets the Network Prefix to a crypto-secure random Unique Local Address (ULA) based on the pattern fdxx:xxxx:xxxx: (RFC 4193).

Return values
kErrorNoneSuccessfully generated a random ULA Network Prefix
kErrorFailedFailed to generate random ULA Network Prefix.
