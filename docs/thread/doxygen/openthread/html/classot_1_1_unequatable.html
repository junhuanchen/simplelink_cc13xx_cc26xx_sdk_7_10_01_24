Detailed Description

template<typename Type>
class ot::Unequatable< Type >

This template class defines an overload of operator !=.

The != implementation uses an existing == overload provided by the Type class.

Users of this class should follow CRTP-style inheritance, i.e., the Type class itself should publicly inherit from Unequatable<Type>.
