Detailed Description

This class implements Resource Record body format of KEY type (RFC 2535).

Member Enumeration Documentation

§ UseFlags

enum UseFlags : uint8_t

This enumeration type represents the use (or key type) flags (RFC 2535 - section 3.1.2).

Enumerator
kAuthConfidPermitted 

Use of the key for authentication and/or confidentiality is permitted.

kAuthPermitted 

Use of the key is only permitted for authentication.

kConfidPermitted 

Use of the key is only permitted for confidentiality.

kNoKey 

No key value (e.g., can indicate zone is not secure).

§ OwnerFlags

enum OwnerFlags : uint8_t

This enumeration type represents key owner (or name type) flags (RFC 2535 - section 3.1.2).

Enumerator
kOwnerUser 

Key is associated with a "user" or "account" at end entity.

kOwnerZone 

Key is a zone key (used for data origin authentication).

kOwnerNonZone 

Key is associated with a non-zone "entity".

kOwnerReserved 

Reserved for future use.
