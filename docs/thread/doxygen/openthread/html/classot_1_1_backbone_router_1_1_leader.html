Detailed Description

This class implements the basic Primary Backbone Router service operations.

Member Enumeration Documentation

§ State

enum State : uint8_t
Enumerator
kStateNone 

Not exist (trigger Backbone Router register its service).

kStateAdded 

Newly added.

kStateRemoved 

Newly removed (trigger Backbone Router register its service).

kStateToTriggerRereg 

Short address or sequence number changes (trigger re-registration). May also have ReregistrationDelay or MlrTimeout update.

kStateRefreshed 

Only ReregistrationDelay or MlrTimeout changes.

kStateUnchanged 

No change on Primary Backbone Router information (only for logging).
