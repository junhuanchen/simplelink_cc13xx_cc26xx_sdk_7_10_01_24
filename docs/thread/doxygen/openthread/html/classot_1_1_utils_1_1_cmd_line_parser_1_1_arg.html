otError ParseAsHexString ( uint8_t *  aBuffer,
uint16_t  aSize 
) const
inline

This method parses the argument as a hex string into a byte array of fixed expected size.

This method returns kErrorNone only when the hex string contains exactly aSize bytes (after parsing). If there are fewer or more bytes in hex string that aSize, the parsed bytes (up to aSize) are copied into the aBuffer and kErrorInvalidArgs is returned.

Parameters
[out]aBufferA pointer to a buffer to output the parsed byte sequence.
[in]aSizeThe expected size of byte sequence (number of bytes after parsing).
Return values
kErrorNoneThe argument was parsed successfully.
kErrorInvalidArgsThe argument is empty or does not contain valid hex bytes and/or not aSize bytes.
