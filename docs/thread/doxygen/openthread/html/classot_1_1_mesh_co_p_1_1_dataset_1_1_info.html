§ GenerateRandom()

Error GenerateRandom ( Instance &  aInstance)

This method populates the Dataset with random fields.

The Network Key, PSKc, Mesh Local Prefix, PAN ID, and Extended PAN ID are generated randomly (crypto-secure) with Network Name set to "OpenThread-%04x" with PAN ID appended as hex. The Channel is chosen randomly from radio's preferred channel mask, Channel Mask is set from radio's supported mask, and Security Policy Flags from current KeyManager value.

Parameters
[in]aInstanceThe OpenThread instance.
Return values
kErrorNoneIf the Dataset was generated successfully.

§ IsSubsetOf()

bool IsSubsetOf ( const Info &  aOther) const

This method checks whether the Dataset is a subset of another one, i.e., all the components in the current Dataset are also present in the aOther and the component values fully match.

The matching of components in the two Datasets excludes Active/Pending Timestamp and Delay components.

Parameters
[in]aOtherThe other Dataset to check against.
Return values
TRUEThe current dataset is a subset of aOther.
FALSEThe current Dataset is not a subset of aOther.
