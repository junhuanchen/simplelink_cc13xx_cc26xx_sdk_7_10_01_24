§ Ownership

enum Ownership : uint8_t

This enumeration represents the message ownership model when a Message instance is passed to a method/function.

Enumerator
kTakeCustody 

This value indicates that the method/function receiving a Message instance should take custody of the message (e.g., the method should Free() the message if no longer needed).

kCopyToUse 

This value indicates that the method/function receiving a Message instance does not own the message (e.g., it should not Free() or Enqueue() it in a queue). The receiving method/function should create a copy/clone of the message to keep (if/when needed).
