This constexpr function checks whether two given C strings are in order (alphabetical order).

This is intended for use from static_assert, e.g., checking if a lookup table entries are sorted. It is not recommended to use this function in other situations as it uses recursion so that it can be constexpr.

Parameters
[in]aFirstThe first string.
[in]aSecondThe second string.
Return values
