§ OT_REST_ARGS

#define OT_REST_ARGS (   ...)    _OT_GET_REST_ARGS(_OT_HAS_ONE_ARG_VS_TWO_OR_MORE(__VA_ARGS__), __VA_ARGS__)

This macro expands to comma separated list of arguments excluding the first one.

If there is only one argument, it expands to empty. If there is more than one argument, it expands to an initial comma followed by all the rest of arguments excluding the first one.

Note
: This macro supports up to 20 arguments.
Parameters
[in]...A list of arguments (MUST contain at least one).
Returns
A comma separated list of arguments excluding the first one.
