This template static method converts an enumeration value to a string using a table array.

Template Parameters
EnumTypeThe enum type.
kLengthThe table array length (number of entries in the array).
Parameters
[in]aEnumThe enumeration value to convert (MUST be of EnumType).
[in]aTableA reference to the array of strings of length kLength. aTable[e] is the string representation of enumeration value e.
[in]aNotFoundThe string to return if the aEnum is not in the aTable.
Returns
The string representation of aEnum from aTable, or aNotFound if it is not in the table.
