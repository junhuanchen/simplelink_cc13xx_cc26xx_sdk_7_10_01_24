§ Find() [1/2]

static const Entry* Find ( const Key &  aKey,
const Entry(&)  aTable[kLength] 
)
inlinestatic

This template method performs binary search in a given sorted table array to find an entry matching a given key.

Note
This method requires the array to be sorted, otherwise its behavior is undefined.
Template Parameters
KeyThe type of Key to search for.
EntryThe table Entry type.
kLengthThe array length (number of entries in the array).

The Entry class MUST provide the following method to compare the entry against a given key.

int Entry::Compare(const Key &aKey) const;

The return value indicates the comparison result between aKey and the entry (similar to strcmp()), i.e., zero means perfect match, positive (> 0) indicates aKey is larger than entry, and negative indicates aKey is smaller than entry.

Note
In the common use of this method as Find(key, kTable) where kTable is a fixed size array, the template types/parameters do not need to be explicitly specified and can be inferred from the passed-in argument.
Parameters
[in]aKeyThe key to search for within the table.
[in]aTableA reference to an array of kLength entries of type Entry
Returns
A pointer to the entry in the table if a match is found, otherwise nullptr (no match in table).

§ IsSorted() [1/2]

static constexpr bool IsSorted ( const Entry(&)  aTable[kLength])
inlinestatic

This template method indicates whether a given table array is sorted based or not.

This method is constexpr and is intended for use in static_asserts to verify that a constexpr lookup table array is sorted. It is not recommended for use in other situations.

Template Parameters
EntryThe table entry type.
kLengthThe array length (number of entries in the array).

The Entry class MUST provide the following static and constexpr method to compare two entries.

constexpr static bool Entry::AreInOrder(const Entry &aFirst, const Entry &aSecond);

The return value MUST be TRUE if the entries are in order, i.e. aFirst < aSecond and FALSE otherwise.

Parameters
[in]aTableA reference to an array of kLength entries on type Entry
Return values
TRUEIf the entries in aTable are sorted.
FALSEIf the entries in aTable are not sorted.
