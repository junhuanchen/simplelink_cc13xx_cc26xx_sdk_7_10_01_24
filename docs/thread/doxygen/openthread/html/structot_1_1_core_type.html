Detailed Description

template<typename FromType>
struct ot::CoreType< FromType >

This structure relates a given public OT type to its corresponding core C++ class/type.

Template Parameters
FromTypeThe public OT type.

Specializations of this template struct are provided for different FromType which include a member type definition named Type to provide the corresponding core class/type related to `FromType.

For example, CoreType<otIp6Address>::Type is defined as Ip6::Address.
