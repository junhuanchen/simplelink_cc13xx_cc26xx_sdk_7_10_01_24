§ operator<()

bool operator< ( const Prefix &  aOther) const

This method overloads operator < to compare two prefixes.

If the two prefixes have the same length N, then the bytes are compared directly (as two big-endian N-bit numbers). If the two prefix have different lengths, the shorter prefix is padded by 0 bit up to the longer prefix length N before the bytes are compared (as big-endian N-bit numbers). If all bytes are equal, the prefix with shorter length is considered smaller.

Parameters
[in]aOtherThe other prefix to compare against.
Return values
TRUEIf the prefix is smaller than aOther.
FALSEIf the prefix is not smaller than aOther.

References ExitNow, Prefix::GetBytes(), Prefix::GetLength(), and ot::Min().
