Detailed Description

This class provides helper methods to convert from a set of uint16_t values (e.g., a non-sequential enum) to string using binary search in a lookup table.

static const char* Lookup ( uint16_t  aKey,
const Entry(&)  aTable[kLength],
const char *  aNotFound = "unknown" 
)
inlinestatic

This static method looks up a key in a given sorted table array (using binary search) and return the associated strings with the key.

Note
This method requires the array to be sorted, otherwise its behavior is undefined.
Template Parameters
kLengthThe array length (number of entries in the array).
Parameters
[in]aKeyThe key to search for within the table.
[in]aTableA reference to an array of kLength entries.
[in]aNotFoundA C string to return if aKey was not found in the table.
Returns
The associated string with aKey in aTable if found, or aNotFound otherwise.
