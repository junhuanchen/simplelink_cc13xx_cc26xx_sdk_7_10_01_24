Detailed Description

template<class Type, uint16_t kPoolSize>
class ot::Pool< Type, kPoolSize >

This template class represents an object pool.

Template Parameters
TypeThe object type. Type should provide GetNext() andSetNext()` so that it can be added to a linked list.
kPoolSizeSpecifies the pool size (maximum number of objects in the pool).
