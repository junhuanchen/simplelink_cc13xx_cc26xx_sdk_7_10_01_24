§ Compare() [1/2]

int Compare ( const Timestamp *  aFirst,
const Timestamp *  aSecond 
)
static

This static method compares two timestamps.

Either one or both aFirst or aSecond can be nullptr. A non-null timestamp is considered greater than a null one. If both are null, they are considered as equal.

Parameters
[in]aFirstA pointer to the first timestamp to compare (can be nullptr).
[in]aSecondA pointer to the second timestamp to compare (can be nullptr).
Return values
-1if aFirst is less than aSecond (aFirst < aSecond).
0if aFirst is equal to aSecond (aFirst == aSecond).
1if aFirst is greater than aSecond (aFirst > aSecond).
