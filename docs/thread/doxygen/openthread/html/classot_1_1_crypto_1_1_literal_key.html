Detailed Description

This class represents a literal key derived from a Key.

Constructor & Destructor Documentation

§ LiteralKey()

LiteralKey ( const Key &  aKey)
explicit

This constructor initializes the LiteralKey from a given Key.

If the aKey is itself represents a literal key the same key buffer pointers are used. If the aKey is a KeyRef then the literal key is extracted. In this case, the extracted key MUST be smaller than kMaxSize.

Parameters
[in]aKeyThe key to convert from.
