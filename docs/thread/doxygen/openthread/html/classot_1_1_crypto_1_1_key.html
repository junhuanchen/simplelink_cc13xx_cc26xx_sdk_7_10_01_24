Detailed Description

This class represents a crypto key.

The Key can represent a literal key (i.e., a pointer to a byte array containing the key along with a key length) or a KeyRef (if OPENTHREAD_CONFIG_PLATFORM_KEY_REFERENCES_ENABLE is enabled).
