Member Enumeration Documentation

§ MessageOrigin

enum MessageOrigin : uint8_t

This enumeration represents an IPv6 message origin.

In case the message is originating from host, it also indicates whether or not it is allowed to passed back the message to the host.

Enumerator
kFromThreadNetif 

Message originates from Thread Netif.

kFromHostDisallowLoopBack 

Message originates from host and should not be passed back to host.

kFromHostAllowLoopBack 

Message originates from host and can be passed back to host.
