Detailed Description

This class implements HMAC-based Extract-and-Expand Key Derivation Function (HKDF) [RFC5869] using SHA-256.

uint8_t *  aOutputKey,

)

This method performs the HKDF Expand step.

The method should be used after a previous Extract call, otherwise its behavior is undefined. In the Expand stage an output key of a given length is derived from the pseudo-random key of Extract stage.

Parameters
[in]aInfoA pointer to buffer containing info sequence.
[in]aInfoLengthThe info length (in bytes).
[out]aOutputKeyBuffer to place the output key (must contain at least aOutputKeyLength bytes).
[in]aOutputKeyLengthThe output key length.
