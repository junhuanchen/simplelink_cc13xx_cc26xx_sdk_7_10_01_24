§ Find() [1/4]

static Error Find ( const Message &  aMessage,
void *  aValue,
uint8_t  aLength 
)
inlinestatic

This static method searches for a TLV with a given type in a message, ensures its length is same or larger than an expected minimum value, and then reads its value into a given buffer.

If the TLV length is smaller than the minimum length aLength, the TLV is considered invalid. In this case, this method returns kErrorParse and the aValue buffer is not updated.

If the TLV length is larger than aLength, the TLV is considered valid, but only the first aLength bytes of the value are read and copied into the aValue buffer.

Template Parameters
TlvTypeThe TLV type to find.
Parameters
[in]aMessageA reference to the message.
[out]aValueA buffer to output the value (must contain at least aLength bytes).
[in]aLengthThe expected (minimum) length of the TLV value.
Return values
kErrorNoneThe TLV was found and read successfully. aValue is updated.
kErrorNotFoundCould not find the TLV with Type aType.
kErrorParseTLV was found but it was not well-formed and could not be parsed.
