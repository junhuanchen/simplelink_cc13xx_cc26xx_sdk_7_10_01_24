§ SetTemporaryChannel()

Error SetTemporaryChannel ( uint8_t  aChannel)

This method sets the temporary IEEE 802.15.4 radio channel.

This method allows user to temporarily change the radio channel and use a different channel (during receive) instead of the PAN channel (from SetPanChannel()). A call to ClearTemporaryChannel() would clear the temporary channel and adopt the PAN channel again. The SetTemporaryChannel() can be used multiple times in row (before a call to ClearTemporaryChannel()) to change the temporary channel.

Parameters
[in]aChannelA IEEE 802.15.4 channel.
Return values
kErrorNoneSuccessfully set the temporary channel
kErrorInvalidArgsThe aChannel is not in the supported channel mask.

bool IsInTransmitState ( void  ) const

This method returns if the MAC layer is in transmit state.

The MAC layer is in transmit state during CSMA/CA, CCA, transmission of Data, Beacon or Data Request frames and receiving of ACK frames. The MAC layer is not in transmit state during transmission of ACK frames or Beacon Requests.
