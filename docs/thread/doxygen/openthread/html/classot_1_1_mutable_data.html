Detailed Description

template<DataLengthType kDataLengthType>
class ot::MutableData< kDataLengthType >

This type represents a generic MutableData which is simply a wrapper over a pointer to a buffer with a given data length.

It inherits from Data but unlike Data which treats its buffer content as immutable, MutableData allows its data buffer content to be changed.

A MutableData instance MUST be initialized (using any of the Init() methods) before calling any other methods (e.g., GetBytes() or GetLength()), otherwise the behavior is undefined.

§ CopyBytesFrom() [1/2]

Error CopyBytesFrom ( const uint8_t *  aBuffer,
LengthType  aLength 
)
inline

This method copies the bytes from a given buffer into the MutableData buffer.

If the current MutableData length is larger than or equal to aLength, then all the bytes are copied from aBuffer into the buffer of MutableData and the MutableData's length is changed to aLength.

If the current MutableData length is smaller than aLength, then the method returns kErrorNoBufs but still copies as many bytes as can fit.

Parameters
[in]aBufferA pointer to a buffer to copy from.
[in]aLengthThe length of aBuffer (number of bytes).
Return values
kErrorNoneSuccessfully copied the bytes into MutableData buffer and adjusted its length.
kErrorNoBufsMutableData buffer cannot fit the given aLength bytes.

This method copies the bytes from an given Data instance into the MutableData buffer.

If the current MutableData length is larger than or equal to the aData length, then all the bytes are copied from aData into the buffer of MutableData and the MutableData's length is adjusted accordingly.
