Detailed Description

Allows to encrypt spinel frames sent between Application Processor (AP) and Network Co-Processor (NCP).

bool SpinelEncrypter::EncryptOutbound ( unsigned char *  aFrameBuf,
size_t  aFrameSize,
size_t *  aFrameLength 
)

Encrypts spinel frames before sending to AP/NCP.

This method encrypts outbound frames in both directions, i.e. from AP to NCP and from NCP to AP.

Parameters
[in,out]aFrameBufPointer to buffer containing the frame, also where the encrypted frame will be placed.
[in]aFrameSizeMax number of bytes in frame buffer (max length of spinel frame + additional data for encryption).
[in,out]aFrameLengthPointer to store frame length, on input value is set to frame length, on output changed to show the frame length after encryption.
Returns
true on success, false otherwise.
