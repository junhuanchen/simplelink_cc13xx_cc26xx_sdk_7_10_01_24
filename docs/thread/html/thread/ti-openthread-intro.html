TI-OpenThread Introduction¶

The SimpleLink CC13xx/CC26xx SDK is part of TI’s SimpleLink MCU platform which offers a single development environment that delivers flexible hardware, software and tool options for customers developing wired and wireless applications. Visit the SimpleLink MCU Platform homepage for more information about the SimpleLink ecosystem.

Thread is an IP-Based mesh networking protocol that runs on IEEE 802.15.4 radios. Thread was designed with one goal in mind: “To create the best way to connect and control products in the home”. For information, see Thread Group Homepage.

OpenThread is an open source implementation of the Thread specification, which includes all Thread networking layers and device roles, as well as Border Router support. For more information, see OpenThread Homepage.

TI-OpenThread is part of the SimpleLink CC13xx/CC26xx SDK, which contains the supported software for the CC13xx or CC26xx. The TI-OpenThread stack integrates the OpenThread stack with the SimpleLink ecosystem. Besides supporting all the OpenThread features, the TI-OpenThread stack has the added benefit of running on top of TI-RTOS7 which inherently adds support for SimpleLink development boards and the SimpleLink development environment. For more information about the TI-OpenThread stack implementation, see Section TI-OpenThread Overview.

Note

Do not use this document as a substitute for the complete Thread protocol specification. For more details, refer to the OpenThread Homepage or the Thread specification from the Thread Group Homepage.
