System Configuration (SysConfig)
================================

SysConfig is a graphical tool for configuring your project in CCS. For a full
documentation on the tool, please see the
`SysConfig SimpleLink Academy <http://dev.ti.com/tirex/explore/node?node=AO3dBAhu05HBWXd.SPxt0g__fc2e6sr__LATEST>`_

To set up SysConfig please see the section :ref:`sysconfig-sysconfig`.
