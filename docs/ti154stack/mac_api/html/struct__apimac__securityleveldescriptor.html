§ securityOverrideSecurityMinimum

bool _apimac_securityleveldescriptor::securityOverrideSecurityMinimum

Indication of whether originating devices for which the Exempt flag is set may override the minimum security level indicated by the Security Minimum element. If TRUE, this indicates that for originating devices with Exempt status, the incoming security level zero is acceptable.
